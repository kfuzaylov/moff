Make your development easy with Moff tools

Modular system

Moff modular system allow to create independent modules with own scope and rich event system.

AMD

AMD - Asynchronous Module Definition API. This is a mechanism to define modules, load it and its dependencies asynchronously.

Event system

Event system for modules communication between each other via events. Which allows to register, sign and trigger events.

API

Moff has a rich API to work with mobile first approach. It has a big number of modules to work with detection, event system and AMD.
