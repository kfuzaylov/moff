- Mobile First Framework v. 1.7.32

Moff is a javascript framework designed to offer best practices for mobile first development. It has rich API to work with browser, files and modularity system. Also it support Asynchronous Module Definition (AMD) which help to control loading of your modules. Data events system allows to load content with the least efforts. And it has pluggable system which allows to create plugins and extend the framework.

Powerful data event helpers

Easily loads some content to specified target.

<a href="additional-content.html" data-load-target="#content-target">Load additional content</a>
<div id="content-target"></div>

Data events + AMD

For example you want to show only main image and load thumbs only when user request them.

<div class="main-image">
	<img src="main-image.png">
</div>
<div class="image-thumbs">
	<a href="thumbs.html"  data-load-target=".image-thumbs" data-load-module="thumbsModule">See more...</a>
</div>
<!-- thumbs.html -->
<div><img src="/images/thumb1.png"></div>
<div><img src="/images/thumb2.png"></div>
<div><img src="/images/thumb3.png"></div>
<div><img src="/images/thumb4.png"></div>
<div><img src="/images/thumb5.png"></div>
<div><img src="/images/thumb6.png"></div>
Moff.amd.register({
	id: 'thumbsModule',
	depend: {
		js: ['bower_components/jquery/dist/jquery.min.js', 'bower_components/slick-carousel/slick/slick.min.js'],
		css:['bower_components/slick-carousel/slick/slick.css']
	},
	file: {
		js: ['javascripts/thumbs.js']
	}
});
// thumbs.js

// jQuery and Slick carousel are already loaded
$('.image-thumbs').slick({
	infinite: true,
	slidesToShow: 3,
	slidesToScroll: 3
});

$('.image-thumbs img').on('click', function() {
	var number = $(this).attr('src').match(/thumb(\d)\.png/)[1];
	$('.main-image img').attr('src', 'images/preview' + number + '.png');
});
